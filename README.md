* 프로세스 생성

```bash
$ node lib/index.js start example/simple-server.js  --name ss
```

```bash
$ node lib/index.js start example/simple-server1.js  --name ss
```

* 프로세스 목록조회

```bash
$ node lib/index.js list
```

* 프로세스 삭제

```bash
$ node lib/index.js delete [index]
```

* 프로세스 로그

```bash
$ node lib/index.js log [index]
```
